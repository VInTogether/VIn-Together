Types of Questions:

  • They will give some code and ask you how many times the loop runs or calculate the time delay created by the code.(mostly these type of questions has three nested loops)
  • they will give some time delay and ask you to write the code to generate that delay (again this has two types of possible questions if the value corresponding to time delay is less than or equal to 255 one code and if it is more than 255 you have to use nested loop )
