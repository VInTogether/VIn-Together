Summary

  • Learn the theory part of dynamic memory allocation with examples and proper syntax.
  • Practice coding problems related to pointers, and also practice questions where they ask the output of the code and pointer arithmetic. (Although they don't ask pointer arithmetic directly, some part of the code contains pointer arithmetic, and you may be asked the output.)
  • Again, if you think you can't deal with coding questions, at least learn the basic syntax or basic code structures for every example. (Even if you don't write the exact code, if you write the correct syntax, some teachers may give marks if they want to make you pass).
