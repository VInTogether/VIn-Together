Strings

Points to remember:

  • Different types of string declarations and what '\0' means in a string.
  • All the string-related operations.
  • Taking input of the string and manipulating the string using string-related operations.

Types of Questions:

  1. Implementation of strcmp, strlen, strcpy, and other string operations without using these functions (i.e., writing code for these functions).
  2. Some coding questions related to strings that involve the use of string-related operations (again, the basic syntax of code, like taking input, is the same for all; at least learn that).
