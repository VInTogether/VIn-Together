Strings

Points to remember:

  • From this part, we can expect theory questions.
  • Definition and advantages of functions.
  • The main difference between function call by reference and call by value.
  • There are some other types of functions, like the classification of functions based on return type and parameters.
  • There is something called inline functions; remember its definition and advantages.

Types of Questions:

  1. Sometimes, they directly ask the difference between call by reference and call by value with examples.
  2. They may give some code that has both call by reference and call by value parameters in a function, ask what the values of the parameters are after the function is called, and ask you to explain (this is again about call by reference and call by value).
  3. There may be a possibility of asking recursive function questions, like Fibonacci series, factorial, or the sum of n numbers.
  4. There is a possibility of questions where they provide some mathematical function and ask you to write a function for that.
