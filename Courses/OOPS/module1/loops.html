Loops

Points to remember:

  • Remember the main difference between while loop and do-while loop (do while loop is a post condition check loop(exit controlled loop) but for and while loops are pre condition check loops(entry controlled loops)) In do while loop the loop body executes atleast once even if the condition is false.
  • Learn how to convert while to for loop and for loop to while loop.
  • Learn how to declare infinite loop using for loop.
  • Remember break and continue statements are loop control statements.
  • Remember the flow charts for all the loop statements.
  • for Loop - for iteration, Count-controlled loop, Definite loop.
  • while Loop - Pre-condition loop, Indefinite loop, Conditional loop.
  • do-while Loop - Post-condition loop, Exit-controlled loop, Guaranteed loop.

Types of Questions:

  • Chances of directlty asking comparisons among different loops.
  • They may give code related to loops and ask how many times loop run and what might he the final output.
  • They may complicate question by involving break and continue statements in the code.
  • They may ask you to write the code logic for a particular scenario like Prime numbers(where you use break statement) .,odd /even numbers(where you use continue statement )like wise application based coding questions are possible.
  • Sometimes direct differences between break and continue statements can be asked.
